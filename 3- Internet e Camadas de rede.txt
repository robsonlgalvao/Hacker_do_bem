Topologias
LAN - Rede Local (Rede doméstica ou Rede empresarial)
para construir devemos verificar:
quantidades de micros, quantidades de switch, distâncias entre os micros, quantidade de notebook, definir acesso sem fio, ligar switch em outros

WAN - Rede de Área Ampla
Rede de longa distância
empresas utilizam VPN(Virtual Private Network)
escritórios compartilhando recursos mesmo não estando no mesmo pais

MAN - Metropolitan Area network
Raio 5KM a 50KM
normalmente ligada por fibra(alugada ou própria)
grande largura de banda
Desvantagem: alto curto

CAN - Campus area network
mesma idéia da MAN Mais barato, gerenciamento centralizado, menor que MAN

SAN - Storage area Network(Rede de armazenamento de alta velocidade)
Normalmente conectada por fibra
Melhor disponibilidade, melhora de desempenho, flexibilidade, disponibilidade, escalabilidade

PAN - Personal area network (Curto alcance)
Cabo - USB, Firewire
Sem fio - Bluetooth,wi-fi,IRDA, Zigbee

BAN - Body area network
utilização - militar, civil
Desvantagens - (qualidade baixa) Dados , sensor, segurança, alto custo

WLAN - Wirelless Local Area Network
Conexão sem fio local, um cabo vem com internet até o roteador que repassa pelo ar.

WMAN - Wirelless Metropolitan Area Network
Necessário AP (access point)

WPAN - Wireless Personal Area Network
Equipamentos wi-fi, bluetooth, zigbee

WWAM - Wirelless Wide Area Network
Redes dos celulares

Topologia de rede

Point to point	- ponto a ponto

2 computadores conectados  por uma via, cabo ou sem fio

BUS		- barramento
( Cabo no inicio até o fim com msm cabo)
se um dos computadores perder conexão, todos os seguintes também perdem conexão

Ring		-  Formato anel
Nessesário 2 placas de rede para cada computador(caro)

Star		- Formato estrela
Dispositivo principal que distribui para os outros(se o central cair cai tudo)


Tree		- Formato Arvore
Varias de estrelas ligadas a outras

Mesh		- Formato Malha
quase inume a falha
todos computadores teriam que ter varias placas de redes, mesma quantidade de computadores


Hybrid		- Formato Hibrido
Junção de varias topologias

MFA - Multiplo fator de altenticação

Ativos e passivos de Rede

Ativos 
Hubs, switches, Roteadores, Placa de Rede, Firewall

Hub - Quando um micro manda informação todos recebem(Broadcast)
ativo - amplificam sinal recebido
passivo - não amplifica

Switch - Recebe uma informação de um micro e manda para o somente para o destinatário atravéz do MAC address
normalmente trabalha na camada 2 do modelo OSI, mas exitem de camada 3.

gerenciáveis 	 - Modelo que dão opções de regras de acesso e comfigurações(vlan)
não gerenciáveis - Modelo simples sem regras de acesso

Passivos
Conectores rj45, réguas de alimentação, patch panel


Roteadores - trabalha com IP, Serve para encaminhar dados entre redes e trabalha na camada 3 do modelo OSI

Placas de Rede  - conecta o computador em outros computadores através da rede

Firewall 	- termo que faz separação de rede interna e externa (Pode ser físico ou software)
tipos
Filtragem de pacotes - Testa os pacotes de acordo com as regras dos ADMs
Os que não atendem são descartados.
filtragem estática(definições de regras manuais)
dinamico (Configurações de tempo de serviço) 
sem estado (mais antigo, protege de malware, trafegos não específico de aplicativos)
estável filtro baseado em extenções 

Firewall Aplicação   - desemvolvidos para verificar aplicativos WAF
Detectar ataques para http/https, escalável
tipos
aplicação de rede, aplicação de host, aplicação na nuvem 

Inspeção de estado   - Avalia o que é padrão o oque está acontecendo

IDS - Sistema de Detecção de Intrusão
Sistema que monitora tráfego de rede e alerta em casos de ameaças
tipos
HIDS - baseado em host, é instalado em cada máquina
NIDS - é na rede monitora a rede toda

IPS - Sistema de prevenção de Intrusão
Sistema que bloqueia o trafego malicioso em resposta à ameaça
HIPS - Baseado em host , é instalado em cada máquina
NIPS - é na rede e monitora a rede toda.
