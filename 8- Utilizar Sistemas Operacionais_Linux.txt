O Linux é um dos sistemas operacionais de código aberto, ou seja, seu código-fonte está disponível para qualquer pessoa modificar, melhorar e distribuir de acordo com os termos das licenças de software livre. 

Desenvolvido pelo engenheiro de software finlandês Linus Torvalds, no início do anos 1990, os Linux cresceu rapidamente em popularidade devido a facilidade no processo de colaboração no desenvolvimento do sistema operacional. 

Linux é o kernel do sistema operacional

Versões

Slackware - Distro mais antigo e tradicional
Debian - Sistema altamente persolalizável seguro e focado em performance
Ubuntu - Baseado no debian - tem tudo do Debian com mais compatibilidade com hardware 
Mint - Baseado no Ubuntu facil migração do windows
Kali linux - Baseado no Debian Distro focada em teste de invasão
ParrotOS - Baseado no Debian Distro focada em teste de invasão com mais recursos de privacidade
Arch Linux - Focada em rapidez estabilidade e segurança, instalação não amigável.

Redhat Enterprise Linux - Baseado no fedora Focado em ambientes corporativos, segurança e suporte para resolução de problemas(não é gratuito).
Fedora - Focada em rápidas atualizações.(extensão dbf)
Android - Sistemas para dispositivos móveis

OBS: Basicamente todos tem versões desktop(Utilização de gráficos) e servidor(Utilização em modo texto)

Sistemas de arquivos

EXT4
Integridade: Journaling/FCSK
criptografia: sim
compactação: não
permissão: sim
Limites: Arquivos 16TB Partição 1EB
Outros Recursos: Compatibilidade Windows MacOs, Cotas em disco,
Estabilidade e velocidade

BTRFS(B-Tree file system)
Integridade:	Copia em gravação(COW)/Auto reparo
Criptografia:	Sim
Compactação:	Sim
Snapshot:	Sim
Permissão:	Sim
Limites:	Arquivos 16EB/Partição 16EB
Outros Recursos: Redimecionamento de volume, RAID, Backup incremental, Desduplicação dedados

JFS - Journaled File System
Integridade 	- Journaling
Criptografia:	- Não
Compactação:	- Não
Snapshots:	- Não 
Permissões	- Sim
Limite: 	- arquivo 4PB/partição 32PB

XFS - High Performance Scalable File System
Integridade:	Journaling
Compactação:	Não
Criptografia:	Não
Snapshots:	Não
Permissões:	Sim
Limites:	Arquivos de 8EB/Partição 16EB
Outros Recursos: Aumentar o tamanho, cotas, leitura e gravação paralela.

ReiserFS
Integridade:	Journaling/FSCK
Criptografia:	Não
Compactação:	Não
Snapshot:	Não
Permissões	Não
Permissões:	Sim
Limites:	Arquivos 8TB/Partição 16TB

ZFS (criado pela oracle)
Integridade:	CoW Checksum/auto reparo
Criptografia:	Sim
Compactação:	Sim
Snapshots:	Sim
Permissões:	Sim
Limites:	Arquivo de 16EB/Partição 256 quatrilhões de Zettabyte
Outros Recursos RAID-Z, Desduplicação de dados
