Componentes do computador

CPU (Central processing unit) Responsável por processar dados e tranformar em informação
Caracteristicas

Frequência(velocidade,clock) -  medido em HZ defice a capacidade do processador ao processar(contas em paralelo)

Núcleos -  existem processadores com um ou vários núcleos

Cache - É o tipo de memória que auxilia no desempenho fazendo diminuir o tempo de transmissão de informações entre processador e outros componentes.

Arquiteturas
X86(32bits) - CISC
X64(64bitis) - CISC
ARM - RISC

Nomeclaturas

AMD Ryzen 7 3700X  ------------ Intel core i7 11700F

X- Alta performance		K- Alto Desempenho/desbloqueado
G- Video integrado		F- Sem video integrado
U- Baixo consumo		S- Edição expecial
3D- Empilamento de cache	U- Eficiência Enegética

Placa-Mãe

Principal componente onde tudo é conectatos

onboard - oq vem incluso na placa
offboard - oq não vem incluso(necessário encaixar)

Socket - Onde coloca o processador
Slot de memória - onde coloca a memória
Slot PCIe -  onde coloca placas de expanção -exemplo placa de vídeo.
Chipset - Um auxliar para o processador
bateria bios - Mantem a BIOs com energia para menter as configurações
BIOS - chip onde fica o programa gerenciador da placa-mãe
Conector 24 pinos - Leva energia para barramentos da placa mãe
Conector 8 pinos -  Leva energia para o processador
Conector SATA - Conecta dispositivos SATA
M.2(SATA/NVMe) - conecta dispositivos M.2(SATA/NVMe)

Tamanhos placa - mãe
ATX 	- grande
mATX 	- Médio(microATX)
ITX 	- pequeno

Memórias

Voláteis(guarda informações somente quando está ligado) mais veloz
Exemplo
RAM(SIMM, DIMM, SDR, DDR, DDR2,DDR3,DDR4,DDR5)
cache(L1,L2,L3)
Video(GDDR3, GDDR4, GDDR5, GDDR6, HBM, HBM2)

Não voláteis (Guarda informações mesmo se desligar)
ROM(PROMs, EPROMs, EEPROMs,Memória flash)
Armazenamento de dados(Pendrive,HD,SSD)

BIOS - Basic input output System  - é um chip na placa-mãe responsável por ferificar todos conponentes de inicialização
UEFI - Unifield Extensible Firmware Interface - É mais moderna tem mais recursos que BIOS normal.

POST (power-on selt-test) 
Verifica: processador, memória, placa de vídeo, USBs, teclados, Discos(HD,SSD,Pendrives), Relógio do sistema, se teve alteração.

Virtualização

Tipo 1(Bare Metal) - 
A virtualização simula o hardware com o Hypervisor, podendo assim distribuir diferentes recursos  para as máquinas virtuais

1/Hardware 2/Hypervisor 3/Maquina Virtual(vm)

exemplos Hypervisor: Proxmox,Hyper-v,KVM,ESXI

Vantagens :Menor consumo de energia, Menor Gastos com hardware, menor espaços em datacenters

Tipo 2  
1/hardware 2/S.O. 3/Hypervisor 4/maquina virtual(VM)
Tem perda de desempenho em relação ao tipo 1
Vantagem: Pode rodas aplicativos junto com Hypervisor, Praticidade, teste de Sistemas diferentes
Exemplos: Virtualbox, VMware player, QEMU

Container - Virtualiza Sistema Operacional
1/hardware 2/S.O. 3/Sistema Container 4/aplicativos

Hypervisor - Virtualiza hardware
Container - Virtualiza Sistema operacional
